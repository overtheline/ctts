# ctts

Node experiments.

Basic webserver with db.

Data visualization.

K-nearest neighbor.

Iris data.

## Development

Developed in OSX environment. Uses Node JS. VSCode is good.

Commands to start:

To initialize:
```text
npm install
```

To run:
```text
mongod
npm run tsc
npm run server
npm run build
```

## Some requirements

- MongoDB
- TypeScript
- React
- Express
